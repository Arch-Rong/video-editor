# 产品概览

## 1. 一句话定位

面向创作者的 **Web 端非线性视频剪辑器**，对标剪映（CapCut）主剪辑体验：导入素材 → 时间轴编排 → 字幕/特效/音频修饰 → 预览 → 导出成片。

## 2. 目标用户

| 角色 | 诉求 |
| --- | --- |
| AIGC 创作者 | 把故事板/生成结果快速拼成可发布视频 |
| 短视频运营 | 多素材剪辑、加字幕配乐、批量出片 |
| 产品/内容同学 | 在浏览器内完成轻度剪辑，无需桌面软件 |
| 嵌入宿主（如 Lumina-Muse） | 在工作台内打开剪辑面板，复用素材与导出能力 |

## 3. 产品形态

- **独立 Web App**：完整项目管理、素材库、导出中心
- **可嵌入 SDK / 包**：宿主传入 `assets + 初始时间轴`，取回 `projectJSON + 导出结果`
- **后续可选**：桌面壳（Electron）以获得更强本地编解码与文件权限

## 4. 核心价值主张

1. **浏览器内可用**：打开即剪，弱依赖本地安装
2. **时间轴一等公民**：多轨、缩放、对齐、软切/硬切、撤销重做
3. **预览即所见图**：播放头、选中片段、属性面板实时联动
4. **导出可靠**：浏览器端组合 + 服务端/FFmpeg 兜底，输出稳定成片
5. **与 AIGC 链路打通**：自然承接生成图/视频/旁白/音乐

## 5. 对标范围（剪映能力切分）

### 5.1 必须具备（产品立住）

- 素材导入（视频 / 图片 / 音频）
- 预览窗口（播放 / 暂停 / 扫移 / 分辨率比例）
- 多轨时间轴（至少 1 视频轨 + 1 音频轨 + 1 字幕轨）
- 片段操作：移动、裁剪手柄、分割、删除、复制
- 画布变换：位置、缩放、旋转、裁切
- 基础转场（淡入淡出等）
- 字幕文本编辑与简单样式
- 音量、淡入淡出
- 撤销 / 重做 / 项目自动保存
- 导出（分辨率 / 帧率 / 格式）

### 5.2 应尽快具备（体验接近剪映）

- 胶片条（filmstrip）预览
- 磁吸对齐、吸附、轨道空白 gap 模式
- 关键帧动画（位移 / 透明度 / 缩放）
- 滤镜与调色基础面板
- 贴纸 / 贴图层
- 变速（匀速 / 简单曲线）
- 模板工程（JSON 模板填充素材）

### 5.3 中长期（差异化 / 专业向）

- 智能字幕（ASR）
- AI 抠像、智能卡点、自动粗剪
- 云端渲染队列与高分辨率导出
- 协作与版本
- 移动端适配或伴侣 App

## 6. 非目标（明确不做或延后）

- 第一期不做完整「影视工业级」合成（多层遮罩、三维跟踪、节点式特效图）
- 不追求首版实现剪映全部商城素材与版权音乐体系
- 不首期做复杂权限/多租户协作（可后续接宿主账号体系）

## 7. 成功标准

| 指标 | MVP 目标 |
| --- | --- |
| 可完成路径 | 用户可在 10 分钟内完成「导入 → 剪辑 → 导出」 |
| 预览流畅性 | 1080p 素材下播放头拖动不出现明显卡死（允许降采样预览） |
| 数据可恢复 | 刷新页面可恢复上次工程（本地或云端） |
| 导出成功率 | 常规 1–3 分钟成片导出失败率 < 5% |
| 嵌入能力 | 宿主可用 props/API 打开编辑器并写回结果 |

## 8. 产品信息架构

```
项目列表
  └─ 编辑器工作台
       ├─ 顶部栏：项目名 / 撤销重做 / 比例 / 导出
       ├─ 左侧：素材库 / 音频 / 文本 / 特效 / 转场 / 滤镜
       ├─ 中央上：预览画布
       ├─ 中央右：属性检查器（随选中对象变化）
       └─ 底部：多轨时间轴 + 播放头 + 缩放
```

## 9. 与现有「剪辑板」的继承关系

Lumina-Muse `chat-main/EditPanel` 已验证的能力应作为 MVP 输入：

- 软切模型：`startTime` / `duration` / `sourceStart`
- 视频轨拖拽重排（DOM 热路径、幽灵预览、插入指示）
- Filmstrip 抽帧与队列限流
- Preview ↔ Timeline 播放头联动
- 导出合并 / trim 调用

本仓库将其升级为正式产品架构，避免继续堆在对话包内。
