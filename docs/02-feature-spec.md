# 功能规格

按模块描述能力与交互约束。优先级标记：

- **P0**：MVP 必做
- **P1**：v1 必备体验
- **P2**：增强能力

---

## 1. 项目管理

| 功能 | 优先级 | 说明 |
| --- | --- | --- |
| 新建工程 | P0 | 指定名称、画布比例（9:16 / 16:9 / 1:1）、帧率（默认 30） |
| 打开 / 列表 | P0 | 本地或云端工程列表 |
| 自动保存 | P0 | 防抖写入；崩溃可恢复 |
| 另存为 / 复制工程 | P1 | |
| 工程设置 | P1 | 分辨率、背景色、时长上限提示 |

## 2. 素材库（Media Library）

| 功能 | 优先级 | 说明 |
| --- | --- | --- |
| 上传视频/图片/音频 | P0 | 拖放上传；显示缩略图、时长、格式 |
| 素材预览 | P0 | 单击预览，不立刻上轨 |
| 拖到时间轴 | P0 | 落到目标轨与时间点；显示落点线 |
| 从宿主注入素材 | P0 | 嵌入模式下接收外部 asset 列表 |
| 素材分组 / 搜索 | P1 | |
| 媒体代理 / 转码入库 | P1 | 超大或特殊编码先转兼容格式 |
| 版权音乐 / 音效库 | P2 | |

### 交互约束

- 图片上视频轨时默认时长 3–5s，可调
- 不支持的编码给出明确错误，并提示转码

## 3. 预览画布（Preview）

| 功能 | 优先级 | 说明 |
| --- | --- | --- |
| 播放 / 暂停 | P0 | 空格快捷键 |
| 播放头联动 | P0 | 时间轴 scrub ↔ 预览 seek |
| 画布比例框 | P0 | 显示安全区可选 |
| 选中片段变换框 | P0 | 拖动手柄改 scale/position |
| 分辨率降级预览 | P1 | 高负载时用代理分辨率 |
| 独奏 / 静音轨预览 | P1 | |
| 全屏预览 | P2 | |

### 交互约束

- scrub 期间使用防抖 seek，避免卡顿
- 选中 overlay（文字/贴纸）时预览区可直接拖动位置

## 4. 时间轴（Timeline）— 产品核心

### 4.1 轨道体系

| 轨道类型 | 优先级 | 说明 |
| --- | --- | --- |
| Video Track | P0 | ≥1 条；支持多视频轨叠画 |
| Audio Track | P0 | ≥1 条；音乐 / 旁白 / 音效可分轨 |
| Text Track | P0 | 字幕 / 标题 |
| Overlay Track | P1 | 贴纸、图片叠加 |
| Effect Track | P2 | 作用于区间的特效 |

### 4.2 片段操作

| 功能 | 优先级 | 说明 |
| --- | --- | --- |
| 选中 / 多选 | P0 / P1 | 单击选中；Shift 多选 P1 |
| 水平移动 | P0 | 改 `timelineStart` |
| 左右裁剪手柄 | P0 | 改 `duration` + `sourceIn/Out`（软切） |
| 分割 | P0 | 播放头处一分为二 |
| 删除 | P0 | Delete / Backspace |
| 复制粘贴 | P1 | |
| 轨道间拖移 | P1 | 同类型轨之间移动 |
| 磁吸 / 对齐线 | P1 | 吸附其他 clip 边缘与播放头 |
| Gap / 涟漪编辑 | P1 | 删除后可选是否闭合空隙 |
| 胶片条 | P0 | 视频轨显示关键帧序列 |
| 音频波形 | P1 | |
| 缩放时间轴 | P0 | 滚轮 / 滑条；固定每秒像素基线 |
| 播放头边缘自动滚动 | P0 | 长轴 scrub 必备 |

### 4.3 视频片段属性

| 属性 | 优先级 |
| --- | --- |
| volume / mute | P0 |
| opacity | P1 |
| transform（x,y,scale,rotation） | P0 |
| crop | P1 |
| speed | P1 |
| blend mode | P2 |

## 10. 导出

| 功能 | 优先级 | 说明 |
| --- | --- | --- |
| 导出面板 | P0 | 分辨率、帧率、码率、格式（mp4） |
| 任务进度 | P0 | |
| 失败重试与日志 | P0 | |
| 仅导出选区 | P1 | |
| 透明 / ProRes 等高阶格式 | P2 | |

### 导出策略（产品层约定）

1. **短工程 / 简单拼接**：浏览器端或轻量服务合成
2. **复杂滤镜 / 字幕 / 多轨**：服务端渲染（FFmpeg / 自研渲染器）
3. 导出结果回写资源库，并通知宿主

## 11. 撤销重做与历史

| 功能 | 优先级 | 说明 |
| --- | --- | --- |
| Undo / Redo | P0 | 快捷键 ⌘Z / ⌘⇧Z |
| 快照栈 | P0 | 以工程不可变快照入栈 |
| 不入栈操作 | P0 | scrub、缩放时间轴、仅 UI hover |

建议：历史快照**不保存** filmstrip blob URL，降低内存（与现有剪辑板实践一致）。

## 12. 快捷键（MVP 子集）

| 键 | 行为 |
| --- | --- |
| Space | 播放 / 暂停 |
| S / Ctrl+B | 分割 |
| Delete | 删除选中 |
| ← / → | 帧级或秒级步进 |
| ⌘Z / ⌘⇧Z | 撤销 / 重做 |
| ⌘S | 手动保存 |
| +/- 或 ⌘滚轮 | 时间轴缩放 |

## 13. 嵌入宿主 API（产品契约）

```ts
interface VideoEditorEmbedProps {
  projectId?: string;
  initialProject?: ProjectJSON;
  assets?: AssetDescriptor[];
  locale?: 'zh' | 'en';
  onChange?: (project: ProjectJSON) => void;
  onExportComplete?: (result: ExportResult) => void;
  onClose?: () => void;
}
```

宿主典型用法：从故事板选中媒体 → 打开编辑器 → 导出后回到会话 / 资产库。
